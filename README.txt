どうぶつ将棋という簡易版の将棋の対戦プログラム
大学で生徒同士で対戦し、上位から順に良い成績を与える　という授業があり、作成したもの
結果　8位/40人、一番良い評定が貰える事が確定するところまでは頑張れた
主に作成したのは対戦中のロジックの部分で、対戦中の描画などは教授側が用意したものを使用した。

対戦会は何度か行われ、最後の対戦会に使用したものが　saigo_taisenkai.py
ミニマックス法とαβ法というものを使用し、n手まで先読みをして、一番良い手を決定する。（nは対戦相手により5~7）

しかし、ミニマックス法だけでは対応できない手が幾つか存在していた。
そこで、特殊な状況を記録し、例外的な手を打てるように jouhou.py　を作成。
特殊な盤面とそのとき打つ手を予め考え記録しておくことで、勝率を向上させた。
